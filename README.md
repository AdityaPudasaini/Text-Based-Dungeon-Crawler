# Text-Based Dungeon Crawler

A simple text-based dungeon crawler game developed in C++.

## Features
- Player vs Enemy combat system
- Boss and common enemies
- Modular OOP design
- Turn-based gameplay

## Project Structure
- Enemy/ → Enemy classes (base, boss, common)
- Player/ → Player logic
- Gameplay/ → Game mechanics
- main.cpp → Entry point

## How to Run

1. Compile using g++:
   g++ main.cpp Enemy/*.cpp Player/*.cpp Gameplay/*.cpp -o game

2. Run:
   ./game
